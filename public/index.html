How everything fits together

High-level architecture

All three repositories work together to form the complete Daily Dine Pro platform:

  • The Spring Boot backend holds core data and business rules, exposing REST APIs.
  • The Flutter app is used daily by customers and mess owners on mobile.
  • The React admin panel is used by platform admins to monitor and configure the system.
